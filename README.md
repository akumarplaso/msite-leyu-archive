# msite-leyu-archive

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或具体网站，仅作为静态 HTML 文件的历史存档与展示用途。

## 项目简介

本仓库用于收集、整理并发布一系列独立的 HTML 页面。这些页面可能来源于不同时期、不同项目的静态内容。通过本仓库，可以方便地查看、回溯或引用这些历史页面。

## 目录说明

```
.
├── README.md        # 本文件
├── pages/           # 存放归档的 HTML 页面目录
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── assets/          # 可选，存放页面引用的资源文件（图片、样式、脚本等）
    └── ...
```

- `pages/`：核心目录，所有归档的 HTML 页面均放置于此。
- `assets/`：若页面需要引用外部资源（如 CSS、JS、图片），可统一存放于此目录，以保持页面完整性。

## 页面归档说明

- 每个 HTML 文件均为完整的独立页面，包含其自身的样式与脚本。
- 页面可能包含不同时期的版本，建议保留原始内容，不做功能性修改。
- 部分页面可能因时间久远而出现样式或功能差异，属于正常现象。

## 维护说明

- 本仓库由维护者不定期更新，用于添加新的归档页面或修正明显错误。
- 欢迎提交 Issue 或 Pull Request，但请确保所提交内容为独立的 HTML 页面，且不包含恶意代码或敏感信息。
- 本仓库不提供在线预览服务，如需查看页面内容，请克隆仓库后本地打开。

## 许可证

本仓库内容仅供存档与学习参考，具体页面可能包含各自的版权信息，请遵守相关法律法规。
